Getting Started

Welcome to the Markdown Editor! This editor allows you to write Markdown text and see the formatted preview in real-time.

Basic Usage

  • Type your Markdown text in the left editor panel
  • See the formatted result instantly in the right preview panel
  • Use the export buttons to save your document in different formats

Export Options

  • PDF Export: Creates a PDF document with your formatted text
  • Word Export: Saves your document in Microsoft Word format
  • HTML Export: Exports the raw HTML of your formatted text

Additional Resources

  • Check out our Markdown Guide for syntax examples
  • Your content is automatically saved in your browser
  • The editor supports standard Markdown syntax plus tables and code blocks
